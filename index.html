  (java-to-bytecode)
  (run-on-jvm))
  ; ==> BRRRRR!!!

JMurmel is a lightweight JVM-based interpreter/ compiler for Murmel, a Lisp dialect inspired by a subset of Common Lisp.

Download JMurmel 1.0 and give it a spin! You may also want to download Murmel 1.0 Language reference for info about the language features that are supported.

See JMurmel Github Repo for current code, information, downloads and source code.
